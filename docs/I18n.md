# Internacionalización (i18n) — ES / EN | Internationalization

---

## Español (México)

### Arquitectura
- `src/lib/i18n.ts` — Diccionario de traducciones ES/EN
- `src/lib/i18n-provider.tsx` — React context con `I18nProvider`, `useT()`, `useI18n()`
- `src/components/language-toggle.tsx` — Dropdown en navbar (visible en móvil y escritorio)

### Detección de Idioma (orden de prioridad)
1. `localStorage.getItem('portfolio-locale')` — preferencia guardada
2. `navigator.language` — idioma principal del navegador/dispositivo
3. `navigator.languages[]` — lista de idiomas preferidos
4. Default: `es` (público objetivo: Latinoamérica)

### Uso en Componentes
```tsx
import { useT } from '@/lib/i18n-provider';
const t = useT();
// t.nav.inicio → "Inicio" (es) | "Home" (en)
```

### Language Toggle
- Ícono de globo terráqueo + nombre del idioma + chevron
- Lista desplegable al pasar el cursor (hover) en escritorio
- Click para abrir/cerrar en móvil
- Siempre visible (escritorio y móvil)
