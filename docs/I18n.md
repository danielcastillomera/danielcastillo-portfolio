## Español (México)

### Arquitectura
- `src/lib/i18n.ts` — Diccionario completo ES/EN
- `src/lib/i18n-provider.tsx` — React context con `I18nProvider`, `useT()`, `useI18n()`

### Claves nuevas en v7.5.0
- `nav.experiencia` — Sección Experiencia Laboral
- `experience.*` — Labels de la sección de experiencia
- `certifications.location` — País y ciudad del certificado
- `certifications.statusLabel` — Etiqueta "Estado"
- `certifications.viewPdf` — Botón del visor PDF
- `skills.exploratory` — Título de la sección exploratoria

### Detección de Idioma (prioridad)
1. `localStorage.getItem('portfolio-locale')` — preferencia guardada
2. `navigator.language` — idioma del navegador
3. `navigator.languages[]` — lista de idiomas
4. Default: `es` (público objetivo: Latinoamérica)
