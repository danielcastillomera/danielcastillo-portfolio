# Internacionalización (i18n) — ES / EN

## Arquitectura
- `src/lib/i18n.ts` — diccionario de traducciones
- `src/lib/i18n-provider.tsx` — React context con `I18nProvider`, `useT()`, `useI18n()`
- `src/components/language-toggle.tsx` — botón de cambio en navbar

## Detección de Idioma
1. `localStorage.getItem('portfolio-locale')` — preferencia guardada
2. `navigator.language` — idioma principal del navegador
3. `navigator.languages` — lista de idiomas preferidos
4. Default: `es` (público objetivo Latino América)

## Uso en Componentes
```tsx
import { useT } from '@/lib/i18n-provider';
const t = useT();
// t.nav.inicio → "Inicio" (es) | "Home" (en)
```

## Añadir Traducciones
Editar `src/lib/i18n.ts` — añadir la clave en ambos objetos `es` y `en`.
