# Internacionalización / Internationalization (i18n)

> `danielcastillo-portfolio` v7.6.2

---

## Español (México)

### Descripción

Sistema de internacionalización ES/EN implementado sin dependencias externas (sin `next-intl`, sin `i18next`, sin `react-i18next`). Usa Context API de React con un hook personalizado.

### Archivos clave

| Archivo | Responsabilidad |
|---------|----------------|
| `src/lib/i18n.ts` | Objeto con todas las traducciones ES y EN. |
| `src/lib/i18n-provider.tsx` | Context provider + hooks `useT()` y `useI18n()`. |

### Uso en componentes

```typescript
import { useT, useI18n } from '@/lib/i18n-provider';

function MiComponente() {
  const t = useT();                    // Objeto de traducciones del locale activo
  const { locale, setLocale } = useI18n(); // locale: 'es' | 'en'

  return <h2>{t.about.title}</h2>;
}
```

### Persistencia

El locale activo se guarda en `localStorage('portfolio-locale')` y se restaura al recargar la página.

### Añadir una traducción nueva

```typescript
// En src/lib/i18n.ts:
export const translations = {
  es: {
    miSeccion: {
      titulo: 'Mi Título.',
      descripcion: 'Descripción en español.',
    },
  },
  en: {
    miSeccion: {
      titulo: 'My Title.',
      descripcion: 'Description in English.',
    },
  },
};
```

### Convenciones de escritura

Todos los textos deben:

1. Comenzar con la primera letra en **mayúscula**.
2. Terminar con un **punto final** (`.`) si son frases o párrafos.
3. Los nombres propios, marcas y acrónimos mantienen su grafía original.

### Idiomas disponibles

| Código | Idioma | Bandera SVG |
|--------|--------|-------------|
| `es` | Español (Ecuador) | SVG inline en `about-me.tsx` |
| `en` | English (United States) | SVG inline en `about-me.tsx` |

## English (United States)

### Description

ES/EN internationalization system implemented without external dependencies (no `next-intl`, no `i18next`, no `react-i18next`). Uses React Context API with a custom hook.

### Key Files

| File | Responsibility |
|------|---------------|
| `src/lib/i18n.ts` | Object with all ES and EN translations. |
| `src/lib/i18n-provider.tsx` | Context provider + `useT()` and `useI18n()` hooks. |

### Usage in Components

```typescript
import { useT, useI18n } from '@/lib/i18n-provider';

function MyComponent() {
  const t = useT();
  const { locale, setLocale } = useI18n();
  return <h2>{t.about.title}</h2>;
}
```

### Persistence

Active locale is saved in `localStorage('portfolio-locale')` and restored on page reload.

### Writing Conventions

All texts must:

1. Start with the first letter **capitalized**.
2. End with a **period** (`.`) if they are sentences or paragraphs.
3. Proper names, brands and acronyms keep their original spelling.

### Available Languages

| Code | Language | Flag SVG |
|------|----------|----------|
| `es` | Spanish (Ecuador) | Inline SVG in `about-me.tsx` |
| `en` | English (United States) | Inline SVG in `about-me.tsx` |
