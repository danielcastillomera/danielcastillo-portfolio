# Daniel Castillo — Portfolio

Portafolio profesional de Daniel Fernando Castillo Mera.
Desarrollador Full-Stack Junior · Universidad de Guayaquil · Ingeniería en Software.

## Stack Tecnológico

| Capa | Tecnología |
|------|-----------|
| Framework | Next.js 15, React 19 |
| Lenguaje | TypeScript |
| Estilos | Tailwind CSS 4 |
| Deployment | Vercel |
| Fuentes | Google Fonts (DM Sans, JetBrains Mono) |

## Características

- **100% responsive** — Mobile-first, adaptable a cualquier dispositivo
- **WCAG 2.1 / 2.2** — Skip link, roles ARIA, prefers-reduced-motion, focus visible, high contrast
- **Cross-browser** — Chrome 80+, Firefox 78+, Safari 14+, Edge 80+
- **SEO optimizado** — Open Graph, JSON-LD structured data, meta tags completos
- **Seguridad** — CSP, HSTS, X-Frame-Options, Referrer-Policy, Permissions-Policy
- **Performance** — Font preconnect, Turbopack dev, image optimization, lazy animations
- **Accesibilidad** — Nivel A mínimo, se apunta a Nivel AAA en contraste y navegación

## Secciones

1. **Hero** — Presentación con partículas animadas y CTA prominente
2. **Tech Marquee** — Banner infinito de tecnologías
3. **Sobre mí** — Bio, educación, stats, capacidades
4. **Proyectos** — Dashboard Enterprise (destacado) + Proyecto Unificado POE
5. **Habilidades** — Barras de progreso animadas por categoría
6. **Contacto** — WhatsApp directo, GitHub, Email
7. **Footer** — Copyright, licencia, redes sociales
8. **WhatsApp Flotante** — Botón fijo con enlace directo

## Instalación

```bash
git clone https://github.com/danielcastillomera/danielcastillo-portfolio.git
cd danielcastillo-portfolio
npm install
npm run dev
```

## Despliegue en Vercel

```bash
git add .
git commit -m "v1.0.0"
git push origin main
```

Vercel detecta automáticamente el proyecto Next.js y lo despliega.
No se requieren variables de entorno.
